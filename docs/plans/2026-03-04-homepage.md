# Homepage Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the default VitePress home page with a custom, Supabase-inspired landing page for Picobase.

**Architecture:** Extend VitePress's default theme by injecting a custom Vue component into the `home-hero-before` slot. The default home layout's empty hero/features sections are hidden with CSS. This keeps the nav, footer, and doc pages untouched while giving full HTML/CSS control over the home page content.

**Tech Stack:** VitePress 1.6.3, Vue 3, CSS custom properties, no external dependencies.

---

### Task 1: Scaffold the custom theme

**Files:**

- Create: `packages/docs/.vitepress/theme/index.ts`
- Create: `packages/docs/.vitepress/theme/style.css`

**Step 1: Create the theme entry point**

```ts
// packages/docs/.vitepress/theme/index.ts
import DefaultTheme from "vitepress/theme";
import { h } from "vue";
import HomeContent from "./HomeContent.vue";
import "./style.css";

export default {
  extends: DefaultTheme,
  Layout() {
    return h(DefaultTheme.Layout, null, {
      "home-hero-before": () => h(HomeContent),
    });
  },
};
```

**Step 2: Create the base CSS file**

This hides the now-empty default VPHero and VPFeatures sections that VitePress renders when `layout: home` is set with no frontmatter content.

```css
/* packages/docs/.vitepress/theme/style.css */
:root {
  --pb-bg: #0a0a0a;
  --pb-bg-card: #141414;
  --pb-border: #242424;
  --pb-text: #f0f0f0;
  --pb-text-muted: #888;
  --pb-accent: #2dd4bf;
  --pb-accent-glow: rgba(45, 212, 191, 0.25);
}

/* Hide the empty default home sections */
.VPHome .VPHero,
.VPHome .VPFeatures {
  display: none;
}

.VPHome {
  padding-bottom: 0 !important;
}
```

**Step 3: Verify VitePress picks up the theme**

Run `pnpm dev` from `packages/docs/` (or repo root with `pnpm --filter @picobase/docs dev`).

Expected: site loads, no console errors, the home page looks the same as before (no visible change yet — HomeContent.vue doesn't exist yet).
